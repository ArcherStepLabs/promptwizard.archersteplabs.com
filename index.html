Prompt Wizard

PromptWizard helps you fix messy copied text before you paste it-fast, simple, user-initiated.

How it works (Quick Settings → Edit → Paste)

  1. Copy text in any app
  2. Open Quick Settings and tap the Prompt Wizard tile
  3. Edit your clipboard text in a full-screen editor
  4. Apply text tools (clean, format, extract)
  5. Close to copy the final result back to your clipboard, then paste manually

Fast text tools

  • Clean up formatting (remove unwanted line breaks, trim spaces, normalise text)
  • Convert case (UPPERCASE, lowercase, Title Case)
  • One-tap formatting helpers for everyday copy/paste

Reusable snippets (Memory Boxes)

  • Save text you reuse often (templates, replies, tags, prompts)
  • Insert snippets quickly with pinned boxes

Built for control and trust

  • User-initiated only: you choose when it runs
  • Works in the foreground while you’re using the editor
  • No clipboard history stored
  • No auto-paste into other apps (you paste manually)

Free vs Pro Free is great for quick edits and light daily use.
Pro (one-time purchase, no subscription) is for power users

  • Unlimited daily usage
  • More Memory Boxes and renaming
  • Pro text tools (including advanced extract/clean options)
  • Extra customisation for pinned actions

Perfect for

  • Cleaning messy text copied from websites, PDFs, chats, and emails
  • Preparing text to send, post, or save
  • Reusing snippets for repeated typing
